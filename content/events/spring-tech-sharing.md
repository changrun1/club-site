# 春季技術分享會

## 活動資訊

- **日期**: 2024年3月15日 (星期五)
- **時間**: 下午2:00 - 5:00
- **地點**: 活動中心二樓會議室
- **費用**: 免費參加

### 主講人：張志明 (科技公司技術總監)
**主題：「現代前端開發趨勢與實戰」**
- React vs Vue vs Angular 比較分析
- 微前端架構設計
- 性能優化最佳實踐

### 主講人：李美玲 (資深後端工程師)
**主題：「雲端原生應用開發」**
- 容器化技術概述
- Kubernetes 實戰經驗
- DevOps 文化與實踐
